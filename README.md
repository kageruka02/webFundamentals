# Web Fundamentals Newsletter Sign-Up

A small static web project demonstrating a responsive newsletter sign-up interface with client-side email validation and a success confirmation state.

## Project Structure

- `index.html` - main page markup for the newsletter sign-up flow
- `styles.css` - styling for the sign-up card, success state, and responsive layout
- `script.js` - form validation and UI state handling for subscribe/dismiss interactions
- `assets/` - image assets used in the UI
- `fonts/` - font license files

## Features

- Email input validation using JavaScript
- Inline error message for invalid email addresses
- Success notification card after valid submission
- Responsive illustration swap for desktop and mobile
- Dismiss button to return to the sign-up form

## How to Run

1. Open `index.html` in a browser.
2. Enter a valid email address into the form.
3. Click `Subscribe to monthly newsletter`.
4. View the confirmation screen and use `Dismiss message` to return.

## Notes

- This is a static front-end project and does not submit data to a backend.
- The project uses Google Fonts via CDN for typography.
